## M7 Treasury System

### Real-Time Fund Management

```
Revenue Generated → M7 Treasury (Holding) → Your Account
```

1. **Aggregate Balance**
   - All revenue streams flow to treasury
   - Real-time balance visibility
   - No minimum holding period

2. **Fund Routing Options**

| Rail | Processing Time | Fee | Best For |
|------|-----------------|-----|----------|
| **Bank Transfer (ACH)** | 1-2 days | 0.5% + $0.25 | Regular transfers |
| **Wire Transfer (SWIFT)** | Same day | 1% + $10 | Faster settlement |
| **Crypto** | 10 mins | Gas only | Jurisdiction-independent |
| **PayPal** | Instant-24h | 2.2% | Quick access |

### Account Configuration

Add multiple bank accounts and crypto wallets:

```json
{
  "bankAccounts": [
    {
      "accountHolder": "Your Name",
      "bankName": "Chase Bank",
      "accountNumber": "****7890",
      "routingNumber": "****0021",
      "isDefault": true
    }
  ],
  "cryptoWallets": [
    {
      "currency": "ethereum",
      "address": "0x742d...bE0",
      "label": "Main Wallet",
      "isDefault": false
    }
  ]
}
```

### Smart Routing Algorithm

M7 automatically selects the best rail based on:
- Transfer amount
- Destination type
- Historical performance
- Fee optimization
